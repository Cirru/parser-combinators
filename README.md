
# Parser Combinators in Clojure

Clojure library to provide simple parser combinators solution.

## Usage


```clojure
[mvc-works/parser-combinators "0.0.1"]
```

## Where's Clojure code?

It's generated from Sepal.clj , try this command:

```bash
lein cirru-sepal
```
